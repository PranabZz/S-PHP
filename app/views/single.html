Largest Contentful Paint (LCP)

Our research indicates that LCP is the strongest predictor of user bounce rates across all studied platforms. Sites achieving an LCP under 2.5 seconds showed 24% lower bounce rates compared to slower-loading counterparts. The following code demonstrates a technique for optimizing image loading to improve LCP:

// Implement priority loading for hero images
const hero = document.querySelector('.hero-image');
if (hero) {
  // Mark as high priority
  hero.setAttribute('fetchpriority', 'high');
  
  // Preload critical image
  const preloadLink = document.createElement('link');
  preloadLink.rel = 'preload';
  preloadLink.as = 'image';
  preloadLink.href = hero.src;
  document.head.appendChild(preloadLink);
}

When implementing these optimizations across our test sites, we observed LCP improvements averaging 1.2 seconds, with particularly significant gains on mobile devices using 4G connections.
